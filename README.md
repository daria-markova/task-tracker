# Task Tracker (Java Console App)

## 
A simple task management system built with Java.
The project demonstrates OOP principles, clean architecture, and Git workflow.

## Features
- Add new tasks
- Mark tasks as in progress
- Mark tasks as done
- Delete tasks
- View all tasks
- Filter tasks by:
    - Status (to do/in progress/done)
    - Priority (low/medium/high).
  
## Tech Stack
- Java 24
- Maven
- OOP (Encapsulation, Abstraction)
- Collections (ArrayList)
- Git / GitHub

## Project Structure
src/main/java
├── app → Main class (UI / menu)
├── model → Task model + enums
├── service → Business logic (TaskService)
